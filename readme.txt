Конфигурационный файл - conifg,py

В поле key указан ключ API IPStack.com, преназначенный для определения страны IP-адреса, уже указанный ключ может обработать 9k IP

В поле filename указано имя файла, можно указать полный путь и, соответственно, другое имя файла, если потребуется

После генерации отчетов (reports) и завершения работы скрипта, отчеты копируются в папку Web. Для визуализиции отчетов в виде веб страницы, требуется ВЕБ-сервер, я использовал локальный сервер OpenServer.

Используется база данных SQLite 3

Для работы скрипта требуется интернет и Python 3.8
Используемые пакеты (ТРЕБУЕТСЯ ДЛЯ РАБОТЫ СКРИПТА):
urllib3 1.25.7 - pip install urllib3
